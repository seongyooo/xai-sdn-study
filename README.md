# XAI + LLM/RAG 기반 Intent-Driven SDN 자동화 공부 레포

XAI(Explainable AI)와 LLM/RAG를 활용한 안전한 Intent-Driven SDN 자동화 시스템을 공부하기 위한 저장소입니다.

단순히 LLM으로 SDN 정책을 생성하는 것에 그치지 않고, Digital Twin으로 사전 검증하고 XAI로 판단 근거를 설명할 수 있는 시스템을 목표로 합니다.

---

## 폴더 구조

```
.
├── papers/       # 읽고 있는 논문 PDF
├── resources/    # 서칭한 참고 자료 및 스크랩
├── notes/        # 직접 작성한 공부 노트 및 로드맵
└── archive/      # 참고한 레포 분석 및 보관 자료
```

## 공부 범위

- SDN (Software-Defined Networking) 기초 및 아키텍처
- OpenFlow 프로토콜 / ONOS 컨트롤러 / Mininet 실습
- LLM / RAG 파이프라인
- XAI (Explainable AI) 기법 — LIME, SHAP, Attention
- Digital Twin 기반 네트워크 사전 검증
- Static Validator 설계 및 구현
